# SOMA - Taller Virtual de Arquitectura

## Visión
Llevar la arquitectura a todos los estratos de la sociedad mediante la eficiencia extrema en el proceso de diseño, utilizando programación orientada a objetos e inteligencia artificial para optimizar lo protocolario y potenciar lo creativo.

## Pilares del Proyecto
- **Eficiencia:** Reducción de costos sin perder la esencia del diseño.
- **Automatización:** Uso de pyRevit y posteriormente Blender (Open Source) para procesos LOD 350.
- **IA:** Red de relaciones y gestión de información compleja.
- **Creatividad:** Herramientas para la extracción de mensajes creativos e intenciones del diseñador.

## Estructura del Repositorio
- `/web`: Desarrollo del sitio oficial (Pagina Web 5.html).
- `/dashboard`: Dashboard de gestión de leads y proyectos.
- `/backend`: Servidor Flask, base de datos SQLite, reportes y proyectos.
- `/aplicaciones_python`: App de Entrevista y herramientas de programa arquitectónico.
- `/metodologia`: Protocolos de diseño organizados por bloque.
- `/recursos_graficos`: Portafolio, plantillas e imágenes de inmersión.
- `/scripts_automatizacion`: Herramientas para pyRevit, Blender y optimización.
