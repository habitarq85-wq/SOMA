# ESCENA 3 — AUTOMATIZACIÓN
*Duración: ~4 segundos*

## Concepto
El código construye solo. Muros, pisos, ventanas se generan automáticamente mientras el arquitecto observa. Lo repetitivo, lo calculable, lo protocolario — todo se programa para liberar al creador.

## Textos en pantalla (aparecen y desaparecen)

| # | Texto | Timing | Posición | Animación |
|---|-------|--------|----------|-----------|
| 1 | `"LO PROTOCOLARIO SE PROGRAMA"` | t=0.0s aparece, t=2.5s se desvanece | Superior (x=50%, y=15%) | Fundido 0.3s in, 0.4s out. Aparece como si fuera un comentario en el código |
| 2 | `"AUTOMATIZAR LO TÉCNICO"` | t=1.5s aparece, t=3.5s se desvanece | Centrado (x=50%, y=45%) | Fundido 0.3s in, 0.4s out. Aparece cuando el modelo se construye más rápido |
| 3 | `"LIBERAR LO CREATIVO"` | t=3.5s aparece, se mantiene hasta transición | Inferior (x=50%, y=80%) | Fundido 0.4s in. Se mantiene hasta el fundido de salida |

**Tipografía:** Sans-serif delgada, color #bc4b21, con un sutil brillo de pantalla (como si fuera texto de terminal). Los textos deben parecer líneas de código comentadas — podrían estar precedidas visualmente por `# ` o `// ` para reforzar la estética de programación.

## Especificaciones de cámara

- **Tipo:** Plano fijo cenital con descenso vertical muy lento
- **Velocidad:** Descenso de 0.5m en 4 segundos (imperceptible, como una grúa bajando en cámara lenta)
- **Lente:** 35mm (gran angular suave para mostrar el código y el modelo en un solo encuadre)
- **Profundidad de campo:** f/5.6 — todo nítido: las manos, el código y el modelo que se construye
- **Altura inicial:** 1.5m sobre las manos (vista cenital)
- **Altura final:** 1.0m sobre las manos (ligeramente más cercano)
- **Movimiento:** Descenso vertical recto, sin desplazamiento horizontal. Eje de cámara perfectamente perpendicular al suelo.

## Iluminación

- **Fuente principal:** Luz fría de pantalla (#8ab8a0, temperatura ~6500K) proveniente del código que fluye
- **Calidad:** Luz ambiental fría con ligero resplandor verde desde abajo
- **Relleno:** Mínimo — el fondo es oscuro, las manos reciben luz solo del código
- **Acento:** Ninguno — toda la luz debe sentirse generada por la interfaz
- **Contraste:** Medio-alto — 5 stops entre el código brillante y el fondo oscuro

## Paleta de colores para esta escena

- **Fondo:** #0a0a0a con textura de pantalla apagada
- **Código fluyendo:** #4a9e6b (verde terminal) y #bc4b21 (terracota para palabras clave)
- **Manos del arquitecto:** Iluminadas desde abajo por el código — tonos fríos con borde verde
- **Modelo construyéndose:** #bc4b21 (terracota wireframe) que se vuelve #c4a878 (beige arena) al completarse
- **Líneas divisorias:** #bc4b21 al 40% (líneas de grid)

## Composición

**Vista cenital:** Dos manos abiertas flotan simétricamente en la parte superior del encuadre. Entre ellas, el código fluye verticalmente como una cascada. En la parte inferior, el modelo arquitectónico se construye. La composición es vertical: código arriba, construcción abajo. Las manos en el centro actúan como bisagra entre el mundo digital (código) y el mundo físico (modelo).

## Elementos visuales clave

- Código Python, definiciones de Grasshopper, ecuaciones paramétricas fluyendo verticalmente
- Modelo arquitectónico construyéndose solo en tiempo real: muros emergen del suelo, pisos se colocan, ventanas se repiten
- Cada línea de código debe generar un elemento constructivo visible
- Las manos del arquitecto están quietas, solo observan — los dedos se mueven ligeramente como dirigiendo una orquesta
- El proceso debe ser rápido, hipnótico, preciso

## Secuencia de construcción

| Tiempo | Elemento construido | Detalle |
|--------|---------------------|---------|
| t=0s a 1.0s | Losa de cimentación | Superficie horizontal emerge del vacío |
| t=1.0s a 2.0s | Muros perimetrales | Muros se extienden desde la losa hacia arriba |
| t=2.0s a 3.0s | Entrepiso y cubierta | Segunda losa aparece, techo se forma |
| t=3.0s a 4.0s | Ventanas y detalles | Aberturas se recortan, fachada se completa |

## Transición a Escena 4
El código se desvanece lentamente. Las manos del arquitecto bajan hasta salir del encuadre. Fundido cruzado de 0.8s donde la luz fría de pantalla se transforma en luz natural cálida de atardecer.
